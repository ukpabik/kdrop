# KDrop

KDrop is a free, open-source, cross-platform app that allows you to quickly and smoothly share files between any device on your local network.

## About 
KDrop provides secure, high-speed data transfer between devices on the same local network. KDrop leverages Multicast DNS for device discovery, and the QUIC protocol for encrypted transport. KDrop ensures your files move quickly and securely without relying on third-party cloud servers. KDrop works on all platforms (Linux, Windows, MacOS), and does not trap you into a singular ecosystem similar to other solutions (AirDrop, etc.).

## Getting Started
To compile KDrop from source code:

### 1. Install Rust

### 2. Clone the `KDrop` repository and `cd kdrop`.

### 3. Run `cargo run --release`.
